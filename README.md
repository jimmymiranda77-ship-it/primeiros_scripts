# Laboratório de Lógica: Primeiros Scripts 🚀

Este repositório centraliza meus primeiros passos práticos na programação, focando no desenvolvimento de lógica, estruturas de repetição e condicionais em diferentes ambientes de execução.

---

## 🎮 Projetos Inclusos

### 1. Jogo da Adivinhação (Python)
Um jogo interativo rodando direto no terminal onde o usuário tenta adivinhar o número secreto gerado pelo sistema.
* **Conceitos aplicados:** Variáveis, laço de repetição `while`, estruturas condicionais `if/elif/else` e geração de números pseudoaleatórios.

### 2. Jogo da Adivinhação (JavaScript/Node.js)
Adaptação da mesma lógica de adivinhação, mas explorando o ambiente do **Node.js** e assincronismo na leitura de dados do terminal.
* **Conceitos aplicados:** Manipulação de módulos nativos (`node:readline/promises`), tratamento de erros com `try/finally`, conversão de tipos de dados (`parseInt`) e controle de fluxo.

---

## 🛠️ Tecnologias e Ferramentas Utilizadas

* **Linguagens:** Python 3, JavaScript (ES6+)
* **Ambiente de Execução:** Node.js v22
